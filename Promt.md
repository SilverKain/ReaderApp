---

# ОСНОВНАЯ ИДЕЯ ПРИЛОЖЕНИЯ

Создать приложение **Markdown Library Reader**, в котором пользователь может:

* загружать `.md` файлы
* хранить их в своей библиотеке
* организовывать библиотеку через **папки и подпапки**
* читать Markdown прямо в приложении
* сохранять **позицию чтения**
* продолжать чтение с того же места

Приложение должно работать:

* на **ПК**
* на **телефонах**
* как **responsive web app**

---

# ТЕХНОЛОГИИ

Используй:

* React (последняя стабильная версия)
* Vite
* TypeScript
* React Router

* TailwindCSS для UI

# ОСНОВНЫЕ ФУНКЦИИ

## 1. Левая панель (Sidebar)

Слева должна быть панель со следующими возможностями:

* загрузить `.md` файл
* создать папку
* создать подпапку
* переименовать папку
* удалить папку
* перетаскивать книги между папками (drag and drop)

Структура библиотеки:

Library
├── Programming
│    ├── React Guide.md
│    └── JS Patterns.md
├── Philosophy
│    └── Stoicism.md
└── Notes
└── MyIdeas.md

Файлы Markdown должны храниться в библиотеке.

---

## 2. Хранение файлов

При загрузке `.md` файла:

* файл читается через File API
* сохраняется в **localStorage или IndexedDB**
* сохраняется мета-информация:

```
{
 id
 title
 folderId
 content
 lastReadPosition
 createdAt
}
```

---

## 3. Чтение Markdown

Основная область приложения — **Markdown reader**.

Функции:

* рендер markdown
* прокрутка
* подсветка кода
* адаптация под мобильный экран

---

## 4. Сохранение позиции чтения

Приложение должно:

* отслеживать scroll position
* сохранять позицию чтения
* при повторном открытии книги
  автоматически прокручивать к месту чтения.

---

## 5. Управление библиотекой

Пользователь может:

* создавать папки
* создавать подпапки
* перемещать книги
* удалять книги
* переименовывать

Структура папок должна быть **древовидной (tree structure)**.
